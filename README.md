# Spring Web API

A Spring Batch Application that imports data from 3 csv to postgreSQL

## Prerequisites

Before you begin, ensure you have the following installed:

- Java 21
- Gradle
- An IDE (e.g., IntelliJ IDEA, Eclipse)

## Getting Started

```bash
./gradlew clean build
./gradlew bootRun
```
